    Project Overview

    Augmented Reality Research Assistant Co-op at Northeastern University's Reality Design Lab (August 2025 - December 2025).

    This project focuses on creating real-time AI-driven camera localization to enable shared spatial alignment across multiple devices without GPS.

    Technical Details

    • Developed a real-time pose estimation server using FastAPI and VGGT (Visual Geometry Grounded Transformer) for multi-device localization.
    • Implemented cross-device spatial synchronization, establishing a shared coordinate system for collaborative AR scenes.
    • Designed debugging and visualization tools in Unity (pose HUDs, coordinate axes, origin spheres) for real-time alignment feedback.
    • Built a full-stack pipeline connecting Python-based AI inference with Unity clients over the network.

    Technologies

    • Server: Python, FastAPI, PyTorch, OpenCV, NumPy
    • Client: C#, Unity, Microsoft HoloLens 2
    • AI Model: VGGT (Visual Geometry Grounded Transformer)
